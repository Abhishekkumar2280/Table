# Report by Year â€“ HTML Table

This project demonstrates the creation of a structured data table using HTML.  
The table displays annual reports by country along with table and figure counts and calculated sample sizes.

## Features
- Country-wise annual report representation
- Use of `rowspan` to group multiple years under one country
- Use of `colspan` for total and summary rows
- Table headers (`thead`) and body (`tbody`) separation
- Custom cell spacing and padding
- Highlighted total row using background color
- Navigation links to additional table pages

## Technologies Used
- HTML5

## Purpose
This project is created for learning and practice of HTML table elements and attributes such as rowspan, colspan, table structure, and basic styling.
